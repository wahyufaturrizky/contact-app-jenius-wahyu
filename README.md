## Installation

Clone down this repository. You will need ` node` and `npm` installed globally on your machine.

Installation:

```bash
  npm install
```

To Run Test Suite:

```bash
  npm test
```

To Start Server:

```bash
  npm start
```

To Visit App:

```bash
  localhost:3000/
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/wahyufaturrizky/contact-app-jenius-wahyu
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
